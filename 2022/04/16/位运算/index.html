位运算

C语言可以对二进制位进行运算，称为位运算

运算符 & | ~ ^ << >>
名称 与 或 非 异或 左移 右移
用处 取1 化1 全反 取反 取乘 取除

与

1 & 0 = 0
0 & 0 = 0
1 & 1 = 1
总结：当第一位与 第二位都为1则运算得1

或

1 | 0 = 1
0 | 0 = 0
1 | 1 = 1
总结：当第一位或 第二位之中有一位为1则运算得1

非

~100 = 011
~001 = 110
~010 = 101
总结：对每一位非运算，将每一位值取反；即0取1，1取0

异或

1 ^ 0 = 1
0 ^ 0 = 0
1 ^ 1 = 0
总结：当第一位与第二位相异则运算得1

左移 右移

1 << 2 = B100 =O4
B100 >> 2 = 1

总结：左移是将一个数的二进制值中的每个位向左移动n位,不够则补零。十进制运算结果相当于乘以2的n次方。右移反之，即结果相当于除以2的次方，不计算余数

tips: 使用左移右移运算，比直接乘除操作执行速度要更快，会提高程序运行速度
