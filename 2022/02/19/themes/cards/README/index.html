Cards | Theme for Hexo

Simple · Swift · Powerful

简洁 · 快速 · 强大

正如其名，「Cards」，一款 单栏样式 & 卡片式设计 的现代 Hexo 主题。

Demo 演示

  • ChrAlpha Blog
  • Theme Cards Demo

Installation 安装

cd hexo
git clone https://github.com/ChrAlpha/hexo-theme-cards.git themes/cards

vi _config.yml
-theme: landscape
+theme: cards

hexo s --debug

你也可以参照主题文档详细了解相关配置。
